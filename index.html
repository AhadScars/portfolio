Project 2 - Blogger Application (Spring Boot-based Blogging Platform)

This project is a simple and secure Blog API developed using Java Spring Boot and PostgreSQL (PgAdmin). It provides functionality for users to create, manage, and organize blog posts while implementing essential features such as user authentication, role-based access, and category management. The API supports JWT authentication for secure login and registration, ensuring that only authorized users can access certain endpoints. Additionally, the system is equipped with role-based access control (Admin & User), allowing for different levels of permissions. Users can easily create, update, and delete blog posts, with robust pagination and sorting mechanisms in place for managing large volumes of posts. The integration of PostgreSQL and PgAdmin ensures a reliable and scalable database solution, while Maven handles the build process for smooth project management. With these features, this Blog API offers a complete, secure, and efficient solution for building and managing a blog platform.View Project

Project 4 - E-Commerce

This E-commerce API allows admin and user roles, with different levels of access to product management and shopping functionalities. Admin users have full control over the product catalog, including the ability to Create, Read, Update, and Delete (CRUD) product details such as description, price, and other attributes. On the other hand, regular users can browse the available products, view their details, and add them to their cart for later checkout. The API ensures secure authentication and authorization using JWT (JSON Web Tokens), allowing for safe and token-based access to all endpoints. Admins can manage the products, while users can easily interact with the product catalog, add items to their cart, and proceed with their purchases—all while ensuring the system's security and scalability.View Project
